    Three differences between web1, web2 and web3:

    1. Web 1.0 considered as read only web with very little interaction where consumer can exchange the information
      together, Web 2.0 considered as read-write web and Web 3.0 considered as read, write & execute web.
    2. Web 1.0 is used for sharing information, Web 2.0 is used for interaction and Web 3.0 is used for analyzing data on
      the web.
    3. In Web 1.0 were millions of users, Web 2.0 billions of users and in Web 3.0 trillions of users.

    Example of the differences, In case user wants to search for drug side-effects:

    • Web 1.0: The user must have the specific URL of the website (static website) with the drug side-effects information.
    • Web 2.0: The user will search for the specific drug side-effects, at a website that provides drugs side-effects.
    • Web 3.0: The user will search in search engine his question (such as google). He will get many results.
